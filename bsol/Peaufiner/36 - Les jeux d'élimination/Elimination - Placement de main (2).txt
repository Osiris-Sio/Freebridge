L'élimination placement de main 

L'élimination - placement de main permet, lorsque les jeux sont équilibrés, d'éviter le choix du sens d'une impasse ou d'obliger l'adversaire à rejouer dans une fourchette.

L'adversaire n'a alors plus que le mauvais choix de rejouer dans la couleur de l'impasse ou dans coupe et défausse.

Le placement de main peut être orientée ou indifférent, selon que l'impasse se fait dans un sens ou dans les 2.

Quand vous observez :

- un grand nombre d'atouts 

- Des couleurs courtes faciles à éliminer

- une ou plusieurs couleurs que vous préférez que les adversaires jouent

- une possibilité de remise en main 

Utiliser alors le jeu d'élimination - placement (plan No 3)
